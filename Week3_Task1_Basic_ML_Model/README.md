# Basic Machine Learning Model Using Python

## Objective

This project demonstrates a complete machine learning workflow using the Titanic dataset.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Workflow

- Load dataset
- Explore data
- Clean missing values
- Feature selection
- Train-test split
- Logistic Regression model
- Performance evaluation

## Results

The Logistic Regression model achieved good classification accuracy on the Titanic dataset.
